📖 General Introduction

Lord of the Tabs is a high-performance, visual tab manager browser extension designed to help you regain control of your browsing session. If you regularly work with dozens of open tabs, suffer from browser slowdowns (high RAM usage), or struggle to find specific pages, Lord of the Tabs is your ultimate tool.

At its core, the extension tracks tab usage, records window groupings, maintains a detailed history of your tabs using high-performance local storage (IndexedDB), and offers powerful tools for batch operations. It operates in two main modes:

🖱️ The Popup

A quick-access panel launched directly from the extension icon. Perfect for rapid searching, filtering by importance rating, switching tabs, muting noise, and configuring extension layout settings.

🖥️ The Dashboard Pro

A full-screen command center overrides your New Tab page. Excellent for advanced analytics, tab sorting, bulk actions, workspace packaging, viewing automatically archived tabs in the Vault, and executing cloud backups.

💡 Did you know?

Lord of the Tabs automatically cleans up unused tab metadata older than 6 months and backs up your session structure locally to keep the extension running fast and lightweight.

🖱️ The Extension Popup

Clicking the extension icon opens the popup panel. It displays a waterfall canvas organizing all your active Chrome windows and their nested tabs.

1. Canvas & Visual Indicators

  • Active Window: Marked by a bright green border. The active tab in the active window has a green outline.
  • Selected Window: Clicking a window header highlights it with a blue border, preparing all its tabs for bulk actions.
  • Views: Toggle between Grid View (compact grid of favicons showing ratings/audio playing status) and List View (expanded list showing full titles and ratings) using the ≣ toggle button.

2. Omnibox Controls

The top toolbar houses essential navigation and search elements:

  • ⚓ Pull URL Button: Grabs the URL of your currently active tab. Left-clicking pulls the domain name (e.g., google.com) into the search bar, while right-clicking pulls the full URL path.
  • 🔍 Search & Exact Toggle: Instantly filters tabs by title or URL. Check Exact to only show exact URL matches.
  • ⭐ Star Filter: Restricts the canvas to tabs with a minimum rating (e.g. 3+ ★).
  • 🔇 Mute All: Instantly mutes audio output across all audible tabs.
  • ⚙️ Settings Toggle: Swings open the slide-out Settings Pane.
  • ↗️ Open Dashboard: Opens the full-screen dashboard in a new tab.

3. Footer Operations (Bulk Actions)

To execute actions on multiple tabs, hold Ctrl (or Cmd on macOS) and click on the tab tiles to select them, then click a footer action:

  • Move: Prompts you to select a destination window to group all selected tabs together.
  • Discard: Hibernates selected tabs to free up RAM. They remain visible on your tab bar, but their processes are put to sleep, reloading only when you click on them.
  • Close: Permanently removes the selected tabs from your browser session.
  • Clear History: Clears metadata history and custom names associated with the selected tabs.

4. Settings Pane Configurations

Open settings using the ⚙️ gear icon to customize your workspace:

  • Icon Size / Density: Choose Normal, Compact, or Tiny grid layouts.
  • Narrow Mode: Forces a strict 3-column layout per window pane.
  • Primary Click Action: Switch between switching to the tab immediately or entering Manager Mode (selecting the tab).
  • Ghost Hover Filtering: Enabling this dims or hides other tabs when you hover over a tab, focusing only on tabs matching the same URL, Domain, or Subdomain.
  • Auto-Fit Layout: Dynamically recalculates the optimal icon size and columns depending on the width of your screen.
  • Open Dashboard on Click: Bypasses the popup entirely, opening the Dashboard immediately when clicking the extension icon.

🖥️ The Dashboard Pro

The Dashboard Pro is a tab management dashboard. It can be set to load automatically when opening a New Tab page or opened manually via the popup.

1. View Navigation

Navigate between four main sections at the top of the dashboard:

📋 Table View

A tabular breakdown of all open tabs. You can sort by columns, filter dynamically, select items, and edit star ratings inline.

📊 Charts View

Provides visual representations of your tab distribution. View tabs grouped by domain, RAM/memory consumption per window, and a domain mixed breakdown.

🗄️ Vault View

Displays archived tabs. Lord of the Tabs automatically archives inactive tabs (not accessed for 48 hours and rated under 4 stars) into the Vault to keep your browser organized.

📂 Saved Workspaces

Lists your packaged workspaces. You can package all selected tabs into a named workspace and restore the bundle later with a single click.

2. Table Grouping Modes

Use the Group By dropdown in the actions header to structure your table layout:

  • None: A flat list of all open tabs.
  • Window: Groups tabs by the window they are currently open in.
  • Domain: Groups tabs by hostname (e.g. all github.com tabs).
  • Clones (URL): Isolates duplicated tabs with identical URLs.
  • Importance: Groups tabs according to their star rating (0 to 5 Stars).

3. Advanced Actions

  • 📦 Bundle Workspace: Packages your selected tabs into a workspace, saving it to IndexedDB, and preparing it for restoration in the Workspaces tab.
  • Merge Duplicates: Scans and merges duplicate tabs, keeping the most recently accessed one active and closing redundant clones.
  • Export CSV: Exports your current filtered list of tabs as a spreadsheet-compatible CSV file.
  • ☁️ Backup to Sheets: Integrates with Google Sheets using Google OAuth2 to upload your tab session structure (Title, URL, Date Opened, Last Access, Importance) into a new spreadsheet in your Google Drive.

⌨️ Shortcuts & Pro Tips

Master Lord of the Tabs with these interactive shortcuts and hidden productivity features:

Shortcut / Action Location Description
Alt + 1 - 5 Popup Rate the importance of a focused tab. Pressing the same number again clears the rating.
Ctrl + Click (or Cmd + Click) Popup & Dashboard Selects individual tabs for batch actions (move, discard, close).
Window Header Click Popup & Dashboard Selects or deselects all tabs within that window group simultaneously.
Right-Click "Pull URL" ⚓ Popup Pulls the full URL of the active tab. A standard left-click pulls only the domain name.
Mouse Hover tooltip Popup Displays complete webpage titles, full URLs, active star ratings, and lineage parent reference.
Ctrl + Shift + Y (or Cmd + Shift + Y) Global / Chrome Panic Mode: Instantly closes all open tabs matching adult/XXX domains, minimizes the original window, and opens a clean new window.
Right-Click "Close" or "Close Selected" Popup & Dashboard Panic Trigger: Triggers the Panic Mode immediately.
Ctrl + Shift + U (or Cmd + Shift + U) Global / Chrome Blacklist Active Domain: Adds the current active tab's domain to the Panic Mode domains list, closes the tab immediately, and shows a system notification.
⚠️ Tab Lineage (Parent-Child tracking)

Tabs opened from links in existing tabs automatically record their parent tab's title and URL. Hovering over a child tab in the Popup reveals this connection in the tooltip, helping you trace your research path.
